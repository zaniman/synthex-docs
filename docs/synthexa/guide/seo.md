# SEO Settings

Optimize your SYNTHEXA site for search engines.

## Site-Level SEO

These settings apply to your entire site.

### Step 1: Open Site Settings

In Framer, go to **Site Settings** → **SEO**

### Step 2: Fill in the Fields

| Setting | Recommended Value |
|---------|-------------------|
| **Site Title** | "Your Agency Name — AI Automation Agency" (50–60 characters) |
| **Description** | Brief description of your services and expertise (150–160 characters) |
| **OG Image** | 1200×630px screenshot of your homepage or branded graphic |
| **Favicon** | 32×32px version of your logo (PNG or SVG) |

### Examples

**Site Title:**
```
Pixometra — AI Automation Agency
```

**Description:**
```
We help businesses automate repetitive workflows and scale operations with AI. 
Custom automation solutions for agencies and enterprises.
```

**OG Image:**
- Screenshot of your homepage hero section
- Make sure it looks good in social preview
- Keep file under 1MB

**Favicon:**
- Square format (32×32px minimum)
- High contrast logo works best
- PNG or SVG format

## Per-Page SEO

Override site settings for individual pages.

### Step 1: Select a Page

In Framer, open the Pages panel and select a page (e.g., /about, /blog).

### Step 2: Click Page Settings

**Right-click the page** → **Settings** → **SEO**

Or look for a **Settings** tab in the properties panel.

### Step 3: Override Metadata

- **Page Title:** Custom title for this page
- **Description:** Custom description for this page
- **Slug:** URL path (auto-generated, but editable)

### Examples

**Homepage:**
- Title: "Pixometra — AI Automation Agency"
- Description: "Custom AI automation solutions to scale your operations."

**/about:**
- Title: "About Us — Pixometra"
- Description: "Meet the team behind Pixometra. Our mission, values, and expertise."

**/blog:**
- Title: "Automation Tips & Insights — Pixometra Blog"
- Description: "Learn how to implement AI automation in your business."

**/blog/:slug (Blog Posts):**
- Auto-set from blog post Title and Summary
- Override individually if needed

## Open Graph (OG) Meta Tags

These control how your site looks when shared on social media.

Framer automatically uses:
- **OG Title** — Your site/page title
- **OG Description** — Your site/page description
- **OG Image** — The image you set (1200×630px)
- **OG Type** — "website" or "article"

When someone shares your site on Twitter, LinkedIn, or Slack, the preview shows your OG image and description.

## Meta Tags for Search Engines

Framer automatically includes:

- `<meta name="description">` — Your site description
- `<meta name="viewport">` — Mobile optimization
- `<link rel="canonical">` — Prevents duplicate content
- `<link rel="icon">` — Favicon

## Mobile Meta Tags

Framer automatically handles:

- **Viewport scaling** — Responsive design
- **Mobile-friendly meta tags**
- **Touch icon** — For bookmarks on mobile

## Structured Data & Schema

For advanced SEO (optional):

1. Check if Framer supports adding custom schema markup
2. Or use a Framer plugin for structured data
3. Examples: Organization schema, LocalBusiness schema, BreadcrumbList

## Best Practices

- **Unique titles** — Each page should have a unique title (50–60 characters)
- **Descriptive descriptions** — Summarize page content in 150–160 characters
- **Keywords naturally** — Include relevant keywords naturally, not forced
- **Fresh content** — Add new blog posts regularly (1–2 per month)
- **Internal links** — Link between related pages and blog posts
- **Mobile-first** — SYNTHEXA is mobile-responsive; test on mobile
- **Fast loading** — Compress images and use WebP format
- **Security** — Always use HTTPS (Framer provides this automatically)

## Monitoring & Analytics

### Framer Analytics

Framer provides basic analytics in **Site Settings** → **Analytics**:
- Page views
- Visitor count
- Traffic sources

### Google Search Console

Set up Google Search Console to monitor:
- Search keywords driving traffic
- Search performance
- Indexing status
- Mobile usability

**To add to Search Console:**
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your domain
3. Verify ownership (add a DNS record)
4. Monitor your search performance

### Google Analytics

For detailed traffic analytics:
1. Create a [Google Analytics account](https://analytics.google.com)
2. Copy your tracking ID from Framer Site Settings → Analytics
3. Configure goals and events
4. Monitor user behavior, conversions, etc.

Once your site is live with proper SEO:

- **Build backlinks** — Share your content to attract incoming links
- **Track analytics** — See what content resonates with visitors
