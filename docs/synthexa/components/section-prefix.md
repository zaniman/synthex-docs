# SectionPrefix

A simple section header prefix component that displays a visual identifier (prefix symbol) and section label. Used across all major sections to maintain consistent branding.

![SectionPrefix](/images/guide/synthexa-110@2x.png)

## How to Edit

To customize SectionPrefix:

1. **Select the component** — Click SectionPrefix directly on the canvas, OR find it in the **Layers** panel and click its name
2. **Open Properties** — The Properties panel (right sidebar) displays all editable fields
3. **Make changes** — Edit any property and see changes update in real-time

## Property Controls

| Property | Type | Description |
|----------|------|-------------|
| **Prefix** | String | Visual prefix symbol (e.g., "//", "→", "--") that appears before the label |
| **Label** | String | Section identifier text (e.g., "deployment", "patterns", "workflows") |

## Global Design Changes

To modify the visual design of SectionPrefix across the entire site:

1. Click **Edit Component** in the Properties panel
2. You'll see the component's **Variant 1 · Primary** variant
3. Edit the visual properties (colors, fonts, spacing, etc.) in this primary variant
4. **All instances of SectionPrefix across the site will automatically update** with your changes

This ensures consistent branding throughout your documentation and website.

## Usage

SectionPrefix is used in:

- **Section headers** — All major content sections (deployment, patterns, workflows, etc.)
- **Feature sections** — To introduce new features or capabilities
- **Page divisions** — To visually separate different topics
- **Consistent branding** — Appears on every page to maintain visual identity

Simply update the Prefix and Label properties to customize each section.
